descripción: En este proyecto realizaremos una introducción a Github dentro de un ambiente colaborativo, realizando cambios, commit, push, crear y manejar ramas y también colaborar utilizando pull requests.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

lista de comandos utilizados : 



git config | para configurar mi usuario

git clone  | descarga el repositorio

git add    | prepara cambios

git commit | guarda los cambios

git push   | sube los cambio

git checkout -b | crea una rama

git checkout main | cambiar a main

git pull   | actualiza el repositorio
